## 🚀 快速开始

### 前置要求
1. LeLamp Runtime 已启动（在 Raspberry Pi 上）
2. LiveKit 服务器已配置
3. 浏览器支持 WebRTC（Chrome、Firefox、Safari、Edge）

### 使用步骤

#### Step 1: 生成 LiveKit Token
在终端运行以下命令生成访问令牌：
```bash
cd lelamp_runtime
uv run python scripts/generate_client_token.py --room lelamp-room --user your-name
```

#### Step 2: 打开 Web Client
在浏览器中打开 `web_client/index.html`：
```bash
# 方法 1: 直接打开文件
open web_client/index.html

# 方法 2: 使用简单 HTTP 服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000/web_client/
```

#### Step 3: 连接设备
1. 粘贴 LiveKit Server URL (例如: `wss://your-project.livekit.cloud`)
2. 粘贴生成的 Access Token
3. 点击"连接设备"按钮

#### Step 4: 开始使用
连接成功后，你可以：
- 👋 点击快捷按钮发送指令
- 📸 使用视觉助手检查作业
- 🎭 让台灯做出有趣的动作
- 💡 调整灯光颜色和效果
- 💬 通过文字聊天与台灯对话

## 📖 功能详解

### 快捷操作面板
预设 4 个常用指令：
- 👋 打招呼
- ⏰ 查看时间
- 😄 讲笑话
- 🎵 唱歌

### 视觉助手面板

#### 拍照识别
**功能**: 使用摄像头识别物体、场景、文字等
**使用方法**:
1. 将物体放在台灯摄像头前
2. 点击"🔍 拍照识别"按钮
3. 等待 AI 分析结果
4. 结果将显示在下方面板中

#### 检查作业
**功能**: AI 批改数学题、语文题等作业
**使用方法**:
1. 将作业本放在台灯摄像头前
2. 点击"📚 检查作业"按钮
3. 等待 AI 批改（通常 3-8 秒）
4. 查看批改结果和正确率

**示例结果**:
```
✅ 第1题: 正确
✅ 第2题: 正确
❌ 第3题: 错误 (5+3 应该等于 8)
✅ 第4题: 正确

正确率: 75% (3/4)
评价: 不错哦，继续加油! 🎉
```

#### 推送飞书
**功能**: 拍照并发送到飞书群组
**前置条件**: 需要在 `.env` 中配置飞书 API
```bash
FEISHU_APP_ID=your_app_id
FEISHU_APP_SECRET=your_app_secret
FEISHU_RECEIVE_ID=your_receive_id
```

## 🔧 技术架构

### 前端技术栈
- **HTML5** - 语义化标签
- **CSS3** - 渐变、动画、Grid/Flexbox 布局
- **Vanilla JavaScript** - 原生 JS，无框架依赖
- **LiveKit JS SDK** - WebRTC 实时通信

### 通信协议
- **WebRTC** - 实时音视频流
- **Data Channel** - 控制指令和数据传输
- **JSON** - 消息格式

### 消息格式

#### 聊天消息
```json
{
  "type": "chat",
  "content": "你好",
  "timestamp": 1234567890
}
```

#### 控制指令
```json
{
  "type": "command",
  "action": "play_recording",
  "params": {
    "recording_name": "nod"
  }
}
```

#### 视觉结果
```json
{
  "type": "vision_result",
  "content": "识别结果: 这是一个苹果",
  "image_base64": "data:image/jpeg;base64,...",
  "timestamp": 1234567890
}
```

#### 摄像头状态
```json
{
  "type": "camera_status",
  "active": true,
  "timestamp": 1234567890
}
```

## 🐛 已知问题

### 限制
1. **Token 手动生成**: 目前需要在终端手动运行脚本生成 Token
   - **解决方案**: 未来版本将集成后端 API 自动生成
2. **无用户账号体系**: 任何人拿到 Token 都能连接
   - **解决方案**: 计划在 v3.0 实现用户认证和设备绑定
3. **浏览器兼容性**: 需要支持 WebRTC 的现代浏览器
   - **支持**: Chrome 74+, Firefox 66+, Safari 12.1+, Edge 79+

### 调试建议
1. **连接失败**: 检查 LiveKit Server URL 是否正确，Token 是否过期
2. **视频不显示**: 检查摄像头权限，确认台灯已启动 LiveKit Agent
3. **指令无响应**: 打开浏览器控制台查看错误日志
4. **动作不执行**: 确认 `lelamp/recordings/` 目录中有对应的 CSV 文件
