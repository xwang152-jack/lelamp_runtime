## 2. 测试环境

### 2.1 开发环境测试

**运行环境**:
- 操作系统: macOS / Linux
- Node.js: 18.x+
- pnpm: 8.x+

**启动命令**:
```bash
cd /Users/jackwang/lelamp_runtime
pnpm dev
```

**访问地址**: http://localhost:5173

### 2.2 需要的准备

1. **LiveKit 测试服务器**
   - 需要 LiveKit Cloud 或本地 LiveKit Server
   - 能够生成 Access Token

2. **LeLamp 后端服务**
   - 运行中的 LeLamp Runtime 后端
   - 连接到同一个 LiveKit 房间

3. **测试 Token 生成**
   ```bash
   python3 scripts/generate_client_token.py
   ```
