# 📖 LeLamp 用户使用指南

## 🌟 欢迎使用 LeLamp 智能台灯

LeLamp 是一个具有个性的 AI 机器人台灯，它能陪您聊天、帮助学习、控制灯光，甚至还能做鬼脸！

---

## 🎮 快速开始

### 1. 连接设备

#### 通过浏览器控制
1. 打开浏览器访问：`http://your-device-ip:5173`
2. 输入服务器地址：`http://your-device-ip:8000`
3. 点击"连接设备"按钮

#### 通过语音控制
1. 确保台灯已启动并听到"系统就绪"提示
2. 直接对台灯说话即可开始对话

### 2. 基本对话

**打招呼**：
```
你：你好
LeLamp：（灯光闪烁）你好呀，小主人！本灯又来陪你啦~
```

**询问时间**：
```
你：现在几点了
LeLamp：（看时间）现在是下午3点45分，怎么，还没写完作业就想着玩了？
```

**讲笑话**：
```
你：讲个笑话
LeLamp：好吧...为什么程序员总是搞混万圣节和圣诞节？因为 Oct 31 == Dec 25！
```

## 🎨 灯光控制

### 调色板
1. 点击"灯光控制"面板
2. 在调色板上选择您喜欢的颜色
3. 点击"设置颜色"按钮
4. 台灯立即变成您选择的颜色

### 预设颜色
- 🔴 **红色** - 热情、活力
- 🟢 **绿色** - 舒适、护眼
- 🔵 **蓝色** - 冷静、专注
- 🟡 **黄色** - 温暖、舒适
- 🟣 **紫色** - 神秘、优雅
- ⚪ **白色** - 明亮、清晰

### 灯光特效
- **呼吸灯** - 柔和的呼吸效果
- **彩虹** - 七彩渐变循环
- **波浪** - 光波流动效果
- **火焰** - 温暖的火焰闪烁
- **星空** - 浪漫的星空闪烁

## 🎭 动作表情

### 自动表情
LeLamp 会根据对话内容自动做出表情：
- **点头/摇头** - 回答是/否问题
- **思考** - 思考复杂问题时
- **兴奋** - 听到好消息时
- **睡觉** - 休息或无聊时
- **跳舞** - 听到音乐或开心时

### 手动控制
1. 点击"动作控制"面板
2. 选择预设动作
3. 点击"播放"按钮
4. LeLamp 立即执行动作

---

## 👁️ 视觉功能

### 作业检查
1. 让 LeLamp 看着作业本
2. 说"帮我检查作业"
3. LeLamp 会自动拍照并分析
4. 告诉您哪些题目做对了，哪些需要改进

### 物体识别
1. 让 LeLamp 看着物体
2. 问"这是什么？"
3. LeLamp 会告诉您物体的名称和特点

### 拍照分享
1. 点击"拍照"按钮
2. LeLamp 会拍一张照片
3. 可以发送到飞书群组分享

---

## ⚙️ 系统设置

### 访问设置
1. 点击右上角"设置"按钮
2. 选择要调整的设置分类

### 主要设置

#### 🤖 LLM 配置
- **模型选择**: 选择 AI 模型
- **API 设置**: 配置 DeepSeek API
- **回复风格**: 调整 AI 性格

#### 📷 摄像头设置
- **分辨率**: 调整拍摄清晰度
- **旋转角度**: 校准摄像头方向
- **翻转模式**: 镜像画面

#### 🎤 语音设置
- **语音速度**: 调整语速快慢
- **音量大小**: 调整音量
- **语音识别**: 优化识别准确度

#### 🔧 硬件设置
- **LED 亮度**: 调整灯光亮度
- **设备端口**: 串口设置
- **电机校准**: 校准舵机

#### 🎭 行为设置
- **问候语**: 设置开机问候
- **降噪功能**: 减少背景噪音
- **动作灵敏度**: 调整反应速度

#### 🎨 界面设置
- **主题选择**: 浅色/深色主题
- **语言设置**: 中文/英文
- **通知设置**: 开启/关闭提醒

## 💡 使用技巧

### 提高识别准确率
1. **环境光线**: 保持适中的光线
2. **说话清晰**: 语速适中，发音清楚
3. **距离适当**: 距离台灯 1-2 米
4. **安静环境**: 减少背景噪音

### 延长续航时间
1. **降低亮度**: 适当降低 LED 亮度
2. **减少动作**: 减少不必要的电机运动
3. **关闭视觉**: 不用时关闭摄像头
4. **休眠模式**: 长时间不用时进入休眠

### 个性化设置
1. **调整性格**: 通过设置调整 LeLamp 的说话风格
2. **自定义问候**: 设置个性化的开机问候语
3. **动作偏好**: 选择喜欢的动作风格
4. **灯光喜好**: 保存常用的灯光设置

## ⚠️ 注意事项

### 安全使用
- 不要遮挡散热孔
- 避免阳光直射
- 不要在潮湿环境使用
- 定期检查电源线

### 正确使用
- 不要强行扭转电机
- 避免从高处跌落
- 不要长时间最大亮度
- 定期清洁镜头和传感器

### 故障处理
如果遇到问题：
1. 检查电源连接
2. 重启设备
3. 查看 Web 界面状态
4. 联系技术支持
