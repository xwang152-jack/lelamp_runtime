## 🔧 真实环境测试

### 准备工作

1. **生成测试 Token**
```bash
cd /Users/jackwang/lelamp_runtime
python3 scripts/generate_client_token.py
```

2. **启动后端服务**
```bash
# 在另一个终端
sudo uv run main.py console
```

3. **获取 LiveKit 连接信息**
- Server URL: 通常从 `.env` 文件中的 `LIVEKIT_URL` 获取
- Token: 从步骤 1 获取

### 端到端测试流程

1. **连接测试**
   - 输入 Server URL 和 Token
   - 点击"连接设备"
   - 验证：成功跳转到控制台页面

2. **对话测试**
   - 发送"你好"消息
   - 观察：后端是否响应
   - 验证：回复消息显示在聊天框

3. **灯光控制测试**
   - 点击预设颜色（如暖红）
   - 观察：LeLamp 设备灯光是否变化
   - 验证：灯光颜色正确

4. **特效测试**
   - 点击"🌈 彩虹"按钮
   - 观察：LeLamp 设备是否显示彩虹特效

5. **断开测试**
   - 点击"断开连接"
   - 验证：返回连接页面

## 📊 性能测试

### 构建性能

```bash
# 清除缓存
rm -rf dist node_modules/.vite

# 首次构建
time pnpm build
# 预期: < 10 秒

# 增量构建（修改一个文件后）
time pnpm build
# 预期: < 3 秒
```

### 运行时性能

打开浏览器开发者工具（F12）:

1. **Network 标签**
   - 刷新页面
   - 查看 `DOMContentLoaded` 时间
   - 预期: < 2 秒

2. **Performance 标签**
   - 录制页面加载
   - 查看帧率（FPS）
   - 预期: 稳定在 60 FPS

3. **Memory 标签**
   - 监控内存使用
   - 执行各种操作
   - 检查是否有内存泄漏
