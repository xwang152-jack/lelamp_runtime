# ⚙️ 前端设置界面快速参考

## 🚀 快速开始

### 访问设置界面
```
1. 启动服务: uv run uvicorn lelamp.api.app:app
2. 打开浏览器: http://localhost:5173
3. 连接设备后点击"设置"按钮
```

## 🔧 常用操作

### 保存配置
1. 修改任意设置项
2. 点击"保存配置"按钮
3. 如需重启，点击"立即重启"

### 重启服务
1. 修改配置后会出现重启提示
2. 点击"立即重启"按钮
3. 确认重启操作
4. 等待 3 秒倒计时
5. 服务自动重启

### 连接 WiFi
1. 点击"扫描网络"
2. 选择要连接的网络
3. 输入密码（如需要）
4. 点击"连接"
5. 等待连接确认

### 断开 WiFi
1. 在"当前连接"卡片
2. 点击"断开连接"按钮
3. 确认断开操作

## 💡 使用技巧

### 快速导航
- 使用左侧菜单快速切换设置分类
- 点击返回按钮回到主界面

### 配置安全
- API Key 会自动隐藏显示
- 密码字段支持显示/隐藏切换
- 敏感配置需要重启后生效

### 状态监控
- 查看顶部状态条了解配置更改状态
- "配置修改后需要重启服务"提示表示有待生效的更改

### 错误处理
- 配置保存失败会显示错误提示
- 网络连接失败会自动重试
- 查看浏览器控制台获取详细错误信息

## 🎯 推荐配置

### 开发环境
```yaml
LLM 模型: deepseek-chat
API 地址: https://api.deepseek.com
视觉功能: 启用
摄像头: 1024x768
LED 亮度: 25
动作冷却: 2.0 秒
```

### 生产环境
```yaml

API 地址: 生产环境 API 地址
视觉功能: 根据需求启用
摄像头: 640x480 (性能优化)
LED 亮度: 50 (视觉效果)
动作冷却: 1.0 秒 (响应速度)
```

### 演示模式
```yaml
LLM 模型: 快速响应模型
视觉功能: 禁用 (提升速度)
摄像头: 低分辨率
LED 亮度: 75 (增强视觉效果)
动作冷却: 0.5 秒 (频繁互动)
```

## 🚨 故障排除

### 设置无法保存
1. 检查后端服务是否运行
2. 查看浏览器控制台错误
3. 确认网络连接正常
4. 尝试刷新页面重新登录

### WiFi 连接失败
1. 确认 WiFi 密码正确
2. 检查网络信号强度
3. 尝试重新扫描网络
4. 重启 WiFi 服务

### 配置未生效
1. 确认已重启服务
2. 检查配置是否正确保存
3. 查看后端日志错误
4. 验证环境变量设置
