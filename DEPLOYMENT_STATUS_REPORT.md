### 🚀 解决方案详情

#### 依赖安装问题
**树莓派Python环境**:
- 系统Python: 3.13.5
- 虚拟环境: Python 3.13.5 (重建中)
- 包管理器: pip 26.0.1
- 软件源: pypi.org + piwheels.org (树莓派优化)

**安装过程**:
```bash
# 1. 清理旧环境
sudo rm -rf .venv

# 2. 创建新环境
python3 -m venv .venv

# 3. 升级pip
.venv/bin/pip install --upgrade pip

# 4. 安装项目依赖
.venv/bin/pip install -e .

# 5. 初始化数据库
python3 -c "from lelamp.database.models import Base; Base.metadata.create_all()"

# 6. 重启服务
.venv/bin/python -m uvicorn lelamp.api.app:app
```
